# REQ-003: GCS Link Manager

## Context

MAVLink v2 is used as the communication protocol between the flight computer and the ground control station. The link must support encryption (AES-256) and handle intermittent connectivity gracefully.

[REQ]
The flight computer software shall manage the encrypted MAVLink session with the ground control station, handling connection establishment, keepalive, and graceful reconnection.
